# Changelog

All notable changes to this project will be documented in this file.

## 0.1.0 - 2026-07-09

### Added

- Initial research-software repository foundation.
- Typed primitives for Weyl coordinates, SU(4) gate records, coupling Hamiltonians, and hardware profiles.
- Gate-mirroring scaffold for near-identity coordinates.
- Compiler-pass interface and pipeline scaffold.
- Benchmark metrics, manifest loading, and report records.
- Examples, tests, documentation, governance files, issue templates, CI workflows, and Netlify-ready website layer.

### Notes

- No benchmark performance claims are included.
- Full compiler, pulse-control, routing, calibration, and backend integration algorithms remain future work.
