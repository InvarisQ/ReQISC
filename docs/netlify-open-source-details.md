## Public benefit

Invaris Quantum ReQISC provides an open, reproducibility-first software foundation for researchers and contributors working on reconfigurable quantum instruction-set abstractions, hardware-aware compiler workflows, and benchmark methodology.

## Open-source purpose

The project turns research architecture into an implementation scaffold: typed primitives, compiler-pass interfaces, hardware-profile records, benchmark manifests, examples, tests, and governance. The goal is to make future research contributions easier to review, reproduce, and extend.

## Technical scope

The project currently includes mathematical primitives, hardware profile abstractions, compiler workflow scaffolding, benchmark metric records, documentation, issue templates, and CI workflows. It does not claim production compiler or hardware-backend maturity.

## Contributor model

Contributors participate through documented issues, research-task proposals, pull requests, tests, examples, and documentation improvements. Maintainers review scope, dependencies, citations, and benchmark claims.

## How Netlify will be used

Netlify will host the public documentation, project pages, contribution guides, benchmark methodology pages, roadmap, and future benchmark reports for the open-source project. The hosted site is the public communication layer for researchers, contributors, and reviewers.

## Current project status

The project is an early-stage scaffold with runnable examples and tests.
