# Calibration Model

Calibration-aware modeling is represented by explicit assumption records.

## Model parameters

Parameters are named assumptions with values, sources, and notes. They are metadata records, not fitted device calibrations.

## Experimental calibration

Experimental calibration requires device access, calibration protocols, versioned data, and uncertainty handling. None of that is implemented in the current package.

## Benchmark assumptions

Benchmark manifests may reference calibration assumptions, but reports must distinguish assumptions from measured values.
