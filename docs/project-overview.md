# Project Overview

## What the project is

Invaris Quantum ReQISC is a Python research-software workbench for implementing and evaluating reconfigurable quantum instruction-set components through explicit software layers.

## What it is not

It is not a production compiler, calibrated hardware-control stack, hardware vendor integration, marketing website, or repository of benchmark claims.

## Why it exists

The project gives researchers and maintainers a clean place to separate mathematical representation, hardware-profile assumptions, compiler-pass interfaces, benchmark manifests, and public documentation.

## Current status

The repository currently provides tested dataclasses, examples, documentation, and scaffolds. Full synthesis, routing, calibration, and backend integrations are planned work.
